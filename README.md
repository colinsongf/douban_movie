# douban movie spider
A douban movie spider using gevent+requests+BeautifulSoup

## Install
```
$ pip install -r requirements.txt
```

## Usage
```
# directly run script
$ python spider.py
# run script with arguments
$ python spider.py --start 1888 --end 2016 --thread 20
```
